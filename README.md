konashi-bme280-i2c
====
Overview

## Description
Swift / Konashi2.0 / [BME280](http://akizukidenshi.com/catalog/g/gK-09421/)をI2C通信でつなぐやり取りの一連です。
I2Cのアドレス登録から値の取得まで、一連の流れの情報がWebにまとまっていなかったのでこちらにあげました。

## Requirement
iOS12.1

## Usage
`pod install`したら動くはず。
処理系はViewControllerにまとまっています。
